Latihan Pembuktian Teorema Himpunan

1. Buktikan: A ∪ A = A
Ambil x ∈ A ∪ A ⇒ x ∈ A atau x ∈ A ⇒ x ∈ A.
Sebaliknya, jika x ∈ A maka jelas x ∈ A ∪ A.
Jadi A ∪ A = A.
2. Buktikan: A ∩ A = A
x ∈ A ∩ A ⇒ x ∈ A dan x ∈ A ⇒ x ∈ A.
Sebaliknya, x ∈ A ⇒ x ∈ A ∩ A.
Maka A ∩ A = A.
3. Buktikan: A ∪ B = B ∪ A
x ∈ A ∪ B ⇒ x ∈ A atau x ∈ B ⇒ x ∈ B atau x ∈ A ⇒ x ∈ B ∪ A.
Jadi komutatif.
4. Buktikan: A ∩ B = B ∩ A
x ∈ A ∩ B ⇒ x ∈ A dan x ∈ B ⇒ x ∈ B dan x ∈ A ⇒ x ∈ B ∩ A.
5. Buktikan: (A ∪ B) ∪ C = A ∪ (B ∪ C)
Gunakan definisi anggota himpunan dan sifat logika OR.
x ∈ (A ∪ B) ∪ C ⇒ (x ∈ A atau B) atau C.
Sama dengan A atau (B atau C).
6. Buktikan: (A ∩ B) ∩ C = A ∩ (B ∩ C)
Gunakan sifat AND pada logika.
x ∈ A dan B dan C berlaku sama.
7. Buktikan: A ∩ (A ∪ B) = A
x ∈ A ∩ (A ∪ B) ⇒ x ∈ A dan (x ∈ A atau B) ⇒ x ∈ A.
8. Buktikan: A ∪ (A ∩ B) = A
x ∈ A ∪ (A ∩ B) ⇒ x ∈ A atau (A dan B) ⇒ x ∈ A.
9. Buktikan hukum De Morgan: (A ∪ B)' = A' ∩ B'
x ∈ (A ∪ B)' ⇒ x tidak di A dan tidak di B ⇒ x ∈ A' dan B'.
10. Buktikan: (A ∩ B)' = A' ∪ B'
x ∈ (A ∩ B)' ⇒ tidak (A dan B) ⇒ tidak A atau tidak B.
