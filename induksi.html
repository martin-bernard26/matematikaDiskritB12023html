Induksi
Matematika

Mathematical Induction
P(1) → P(2) → P(3) →  P(n) → P(n+1)

Induksi matematika adalah teknik pembuktian yang elegaan dan kuat — membuktikan sebuah pernyataan berlaku untuk semua bilangan asli hanya dengan dua langkah sederhana. Seperti barisan domino: jika yang pertama jatuh, dan setiap domino menjatuhkan yang berikutnya, maka semua domino pasti akan jatuh.

01 — Konsep Dasar

Apa Itu Induksi Matematika?

Induksi matematika adalah metode pembuktian untuk menegakkan bahwa suatu proposisi P(n) berlaku untuk setiap bilangan bulat positif n ≥ n₀. Metode ini didasarkan pada prinsip urutan baik bilangan asli.

🎯

Tujuan

Membuktikan pernyataan yang melibatkan variabel bilangan bulat positif n tanpa harus menguji satu per satu.

⚙️

Mekanisme

Cukup dua langkah: buktikan untuk n awal (basis), lalu buktikan jika berlaku untuk k, maka berlaku untuk k+1 (induktif).

🔗

Analogi

Seperti tangga tak hingga: jika bisa naik anak tangga pertama, dan dari setiap anak tangga bisa ke anak tangga berikutnya — maka bisa naik ke mana saja.

📐

Penggunaan

Membuktikan rumus deret, ketaksamaan, sifat keterbagian, rumus kombinatorika, dan teorema rekursi.

∀

Prinsip Induksi Matematika

Prinsip ini menyatakan bahwa:

Misalkan P(n) adalah proposisi yang melibatkan bilangan bulat positif n.
Jika terpenuhi:
  ① P(1) benar                     ← Basis / Langkah Awal
  ② Untuk setiap k ≥ 1, jika P(k) benar,
     maka P(k+1) juga benar      ← Langkah Induktif

Maka P(n) benar untuk semua n ∈ ℤ⁺
⚠️ Perhatian Penting

Kedua langkah harus dibuktikan. Hanya membuktikan basis saja (tanpa langkah induktif) tidak cukup untuk menyimpulkan pernyataan berlaku untuk semua n. Begitu pula sebaliknya.

02 — Prosedur

Langkah-Langkah Pembuktian

Berikut adalah prosedur sistematis yang harus diikuti dalam setiap pembuktian dengan induksi matematika:

1
Tuliskan Pernyataan P(n)
Nyatakan secara eksplisit apa yang ingin dibuktikan, dalam bentuk proposisi P(n) yang melibatkan variabel n. Pastikan domain n jelas (n ≥ 1, n ≥ 0, dsb.).
2
Langkah Basis — Buktikan P(1)
Substitusikan n = 1 (atau nilai awal yang sesuai) ke pernyataan P(n) dan verifikasi kebenarannya secara langsung melalui perhitungan.
3
Hipotesis Induktif — Asumsikan P(k) Benar
Asumsikan P(k) benar untuk sembarang bilangan bulat positif k yang tetap. Ini disebut Hipotesis Induktif — sebuah asumsi, bukan hasil yang sudah terbukti sendiri.
4
Langkah Induktif — Buktikan P(k) ⟹ P(k+1)
Dengan menggunakan hipotesis induktif (P(k) benar), buktikan bahwa P(k+1) juga harus benar. Ini adalah inti dari pembuktian — gunakan aljabar, manipulasi ekspresi, atau argumen logis.
5
Kesimpulan
Karena P(1) benar (basis) dan P(k) ⟹ P(k+1) (induktif), maka berdasarkan Prinsip Induksi Matematika, P(n) benar untuk semua n ∈ ℤ⁺. □
03 — Contoh 1

Rumus Jumlah Bilangan Asli

Salah satu contoh paling klasik induksi matematika adalah pembuktian rumus jumlah bilangan asli pertama.

Teorema Deret Aritmetika — Jumlah 1 hingga n

Buktikan bahwa untuk setiap bilangan bulat positif n berlaku:
1 + 2 + 3 + … + n = n(n+1)/2

📌 Langkah 1 — Pernyataan
P(n) : 1 + 2 + 3 + … + n = n(n+1)/2
✅ Langkah 2 — Basis (n = 1)
Ruas kiri : 1
Ruas kanan : 1(1+1)/2 = 2/2 = 1
Ruas kiri = Ruas kanan ✓ → P(1) benar
🔵 Langkah 3 — Hipotesis Induktif
Asumsikan P(k) benar untuk suatu k ∈ ℤ⁺:
1 + 2 + 3 + … + k = k(k+1)/2
🔧 Langkah 4 — Langkah Induktif [Buktikan P(k+1)]
Harus dibuktikan: 1 + 2 + … + k + (k+1) = (k+1)(k+2)/2

Ruas kiri:
= (1 + 2 + … + k) + (k+1)
= k(k+1)/2 + (k+1) ← gunakan hipotesis induktif
= (k+1) · [k/2 + 1]
= (k+1) · [(k + 2)/2]
= (k+1)(k+2)/2 ✓
∎  Terbukti — P(n) benar untuk semua n ∈ ℤ⁺
💡 Verifikasi Cepat

n = 10 → Jumlah = 1+2+…+10 = 55  |  Formula: 10×11/2 = 55 ✓
n = 100 → Jumlah = 5050  |  Formula: 100×101/2 = 5050 ✓ (rumus Gauss yang terkenal!)

04 — Contoh 2

Sifat Keterbagian

Induksi matematika juga sangat berguna untuk membuktikan sifat keterbagian dalam teori bilangan.

Teorema Keterbagian — 3 membagi (n³ − n)

Buktikan bahwa untuk setiap bilangan bulat positif n, ekspresi n³ − n habis dibagi 3.

📌 Langkah 1 — Pernyataan
P(n) : 3 | (n³ − n)   [3 membagi n³ − n]
✅ Langkah 2 — Basis (n = 1)
1³ − 1 = 1 − 1 = 0
3 | 0 ✓ → P(1) benar
🔵 Langkah 3 — Hipotesis Induktif
Asumsikan P(k) benar: 3 | (k³ − k)
Artinya: k³ − k = 3m untuk suatu bilangan bulat m
🔧 Langkah 4 — Langkah Induktif
Perlu buktikan: 3 | ((k+1)³ − (k+1))

(k+1)³ − (k+1)
= (k³ + 3k² + 3k + 1) − (k+1)
= k³ + 3k² + 3k + 1 − k − 1
= k³ − k + 3k² + 3k
= (k³ − k) + 3k(k + 1)
= 3m + 3k(k+1) ← gunakan hipotesis induktif
= 3[m + k(k+1)]
= 3 × bilangan bulat ✓
∎  Terbukti — n³ − n habis dibagi 3 untuk semua n ∈ ℤ⁺
💡 Fakta Menarik

n³ − n = n(n−1)(n+1) = (n−1)·n·(n+1) adalah hasil kali tiga bilangan bulat berurutan. Tiga bilangan berurutan selalu mengandung tepat satu kelipatan 3 — sehingga hasilnya selalu habis dibagi 3!

05 — Contoh 3

Pembuktian Ketaksamaan

Induksi matematika juga dapat digunakan untuk membuktikan ketaksamaan yang melibatkan ekspresi eksponensial.

Teorema Ketaksamaan Bernoulli — 2ⁿ ≥ n + 1

Buktikan bahwa untuk setiap bilangan bulat positif n berlaku: 2ⁿ ≥ n + 1

📌 Langkah 1 — Pernyataan
P(n) : 2ⁿ ≥ n + 1
✅ Langkah 2 — Basis (n = 1)
Ruas kiri : 2¹ = 2
Ruas kanan : 1 + 1 = 2
2 ≥ 2 ✓ → P(1) benar
🔵 Langkah 3 — Hipotesis Induktif
Asumsikan P(k) benar: 2ᵏ ≥ k + 1
🔧 Langkah 4 — Langkah Induktif
Perlu buktikan: 2ᵏ⁺¹ ≥ (k+1) + 1 = k + 2

2ᵏ⁺¹ = 2 · 2ᵏ
     ≥ 2 · (k + 1) ← gunakan hipotesis induktif: 2ᵏ ≥ k+1
     = 2k + 2
     = (k + 2) + k
     ≥ k + 2             ← karena k ≥ 0
Jadi: 2ᵏ⁺¹ ≥ k + 2 ✓
∎  Terbukti — 2ⁿ ≥ n + 1 untuk semua n ∈ ℤ⁺
∑

Ringkasan Perbandingan Contoh

# Pernyataan Tipe Formula / Klaim

2 Keterbagian n³−n Sifat keterbagian 3 | (n³ − n)

06 — Lanjutan

Induksi Matematika Kuat

Induksi matematika kuat adalah variasi dari induksi biasa. Pada langkah induktifnya, hipotesis yang digunakan lebih kuat: diasumsikan P(n) benar untuk semua nilai dari basis hingga k, bukan hanya P(k) saja.

Prinsip Induksi Kuat:

Jika:
  ① P(1) benar (basis)
  ② Untuk setiap k ≥ 1, jika P(1), P(2), …, P(k) semua benar,
     maka P(k+1) juga benar

Maka P(n) benar untuk semua n ∈ ℤ⁺
🔄

Induksi Biasa

Hipotesis: P(k) benar.
Cukup untuk membuktikan P(k+1) dari satu nilai sebelumnya.

💪

Induksi Kuat

Hipotesis: P(1), P(2), …, P(k) semua benar.
Berguna saat P(k+1) bergantung pada lebih dari satu nilai sebelumnya.

🎯

Kapan Digunakan?

Rekursi (seperti Fibonacci), pembuktian teorema faktorisasi prima, analisis algoritma rekursif.

Contoh Penerapan: Bilangan Fibonacci

Didefinisikan F(1) = 1, F(2) = 1, F(n) = F(n−1) + F(n−2) untuk n ≥ 3. Pembuktian sifat Fibonacci sering memerlukan hipotesis untuk F(k) dan F(k−1), sehingga induksi kuat lebih tepat digunakan.

Misalnya, membuktikan bahwa F(n) < 2ⁿ untuk semua n ≥ 1 memerlukan asumsi F(k−1) < 2ᵏ⁻¹ dan F(k) < 2ᵏ secara bersamaan — ini adalah ciri khas induksi kuat.
