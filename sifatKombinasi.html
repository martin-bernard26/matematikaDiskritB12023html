01
Definisi Kombinasi
Fondasi notasi dan formula dasar

Koefisien binomial \(\binom{n}{k}\) — dibaca "n pilih k" — menyatakan banyaknya cara memilih \(k\) objek dari \(n\) objek yang tersedia tanpa memperhatikan urutan.

\[\binom{n}{k} = \frac{n!}{k!\,(n-k)!}\]
Catatan Penting

Notasi lain yang sering digunakan: \(C(n,k)\), \(C_n^k\), atau \({}_nC_k\). Semua menyatakan hal yang sama.

Syarat domain: \(n \geq 0\), \(0 \leq k \leq n\), \(n, k \in \mathbb{Z}\). Jika \(k < 0\) atau \(k > n\), maka \(\binom{n}{k} = 0\).

Contoh

\(\binom{5}{2} = \dfrac{5!}{2!\cdot 3!} = \dfrac{5 \times 4}{2 \times 1} = 10\)

Artinya: ada 10 cara memilih 2 orang dari 5 orang yang ada.

02
Identitas-Identitas Dasar Kombinasi
Klik setiap kartu untuk melihat bukti dan contoh
ID-1 Identitas Simetri (Refleksi) ›
\[\binom{n}{k} = \binom{n}{n-k}\]
Bukti Aljabar

\(\binom{n}{n-k} = \dfrac{n!}{(n-k)!\,k!} = \dfrac{n!}{k!\,(n-k)!} = \binom{n}{k}\) ✓

Intuisi Kombinatorial

Memilih \(k\) objek yang diambil ekuivalen dengan memilih \(n-k\) objek yang ditinggalkan. Keduanya menghasilkan pemilihan yang sama.

Contoh

\(\binom{8}{3} = \binom{8}{5} = 56\)

ID-2 Identitas Batas (Trivial) ›
\[\binom{n}{0} = \binom{n}{n} = 1\]
Bukti

\(\binom{n}{0} = \dfrac{n!}{0!\,n!} = 1\) karena \(0! = 1\).

\(\binom{n}{n} = \dfrac{n!}{n!\,0!} = 1\).

Makna

Hanya ada 1 cara memilih 0 elemen (tidak memilih apapun) dan 1 cara memilih semua n elemen.

ID-3 Identitas Penjumlahan Pascal (Rekursi Pascal) ›
\[\binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}\]
Bukti Kombinatorial

Pilih 1 elemen khusus dari \(n\) elemen, sebut \(x\).

• Subhimpunan yang memuat \(x\): pilih \(k-1\) dari \(n-1\) sisanya → \(\binom{n-1}{k-1}\)

• Subhimpunan yang tidak memuat \(x\): pilih \(k\) dari \(n-1\) sisanya → \(\binom{n-1}{k}\)

Karena kedua kasus saling terpisah (mutually exclusive) dan lengkap: total = \(\binom{n-1}{k-1} + \binom{n-1}{k}\) ✓

Contoh Numerik

\(\binom{6}{3} = \binom{5}{2} + \binom{5}{3} = 10 + 10 = 20\) ✓

Identitas ini adalah dasar dari konstruksi Segitiga Pascal.

ID-4 Identitas Penyerapan (Absorption Identity) ›
\[k\binom{n}{k} = n\binom{n-1}{k-1}\]
Bukti Aljabar

\(k\cdot\dfrac{n!}{k!\,(n-k)!} = \dfrac{n!}{(k-1)!\,(n-k)!} = n\cdot\dfrac{(n-1)!}{(k-1)!\,(n-k)!} = n\binom{n-1}{k-1}\) ✓

Contoh

\(3\binom{7}{3} = 7\binom{6}{2}\)

\(3 \times 35 = 7 \times 15 = 105\) ✓

ID-5 Identitas Jumlah Baris (Vandermonde – Baris) ›
\[\sum_{k=0}^{n} \binom{n}{k} = 2^n\]
Bukti via Teorema Binomial

Substitusi \(x=1\) dan \(y=1\) ke dalam \((x+y)^n = \displaystyle\sum_{k=0}^n \binom{n}{k} x^{n-k} y^k\):

\(2^n = \displaystyle\sum_{k=0}^n \binom{n}{k}\) ✓

Bukti Kombinatorial

Setiap subhimpunan dari himpunan \(n\) elemen terhitung tepat sekali. Banyak subhimpunan adalah \(2^n\) (setiap elemen masuk atau tidak).

Contoh: n = 4

\(\binom{4}{0}+\binom{4}{1}+\binom{4}{2}+\binom{4}{3}+\binom{4}{4} = 1+4+6+4+1 = 16 = 2^4\) ✓

ID-6 Identitas Alternating Sum ›
\[\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0 \quad (n \geq 1)\]
Bukti via Teorema Binomial

Substitusi \(x=1\) dan \(y=-1\):

\((1-1)^n = 0^n = \displaystyle\sum_{k=0}^n (-1)^k\binom{n}{k} = 0\) ✓ (untuk \(n \geq 1\))

Contoh: n = 4

\(1 - 4 + 6 - 4 + 1 = 0\) ✓

ID-7 Identitas Vandermonde ›
\[\binom{m+n}{r} = \sum_{k=0}^{r} \binom{m}{k}\binom{n}{r-k}\]
Bukti Kombinatorial

Pilih \(r\) dari \(m+n\) elemen (grup A berisi \(m\) dan grup B berisi \(n\) elemen).

Untuk setiap \(k\): pilih \(k\) dari grup A dan \(r-k\) dari grup B → \(\binom{m}{k}\binom{n}{r-k}\)

Jumlahkan untuk \(k = 0, 1, \ldots, r\) menghasilkan semua cara → \(\binom{m+n}{r}\) ✓

Kasus Khusus: m = n = r

\(\binom{2n}{n} = \displaystyle\sum_{k=0}^{n}\binom{n}{k}^2\)

Contoh: \(\binom{6}{3} = 1^2+9+25+25+9+1 = 20\)?

\(\binom{4}{0}^2+\binom{4}{1}^2+\binom{4}{2}^2 = 1+16+36+16+1 = 70 = \binom{8}{4}\) (n=4) ✓

ID-8 Identitas Jumlah Diagonal (Hockey Stick) ›
\[\sum_{i=0}^{r} \binom{n+i}{i} = \binom{n+r+1}{r}\]
\[\text{Bentuk lain: }\sum_{i=k}^{n}\binom{i}{k} = \binom{n+1}{k+1}\]
Bukti via Induksi Matematika

Basis: \(r=0\): \(\binom{n}{0} = 1 = \binom{n+1}{0}\) ✓

Langkah Induksi: Asumsikan berlaku untuk \(r\), tunjukkan untuk \(r+1\):

\(\displaystyle\sum_{i=0}^{r+1}\binom{n+i}{i} = \binom{n+r+1}{r} + \binom{n+r+1}{r+1} = \binom{n+r+2}{r+1}\) (oleh ID-3) ✓

Contoh: n=2, r=3

\(\binom{2}{0}+\binom{3}{1}+\binom{4}{2}+\binom{5}{3} = 1+3+6+10 = 20 = \binom{6}{3}\) ✓

Dinamakan "hockey stick" karena pola diagonal di Segitiga Pascal membentuk gagang + kepala tongkat hoki.

ID-9 Identitas Jumlah Genap = Jumlah Ganjil ›
\[\sum_{k \text{ genap}}\binom{n}{k} = \sum_{k \text{ ganjil}}\binom{n}{k} = 2^{n-1}\]
Bukti

Dari ID-5: \(\displaystyle\sum_{k=0}^n\binom{n}{k} = 2^n\) dan dari ID-6: \(\displaystyle\sum_{k=0}^n(-1)^k\binom{n}{k} = 0\)

Tambahkan: \(2\displaystyle\sum_{k \text{ genap}}\binom{n}{k} = 2^n\), jadi masing-masing = \(2^{n-1}\) ✓

Contoh: n = 5

Genap: \(\binom{5}{0}+\binom{5}{2}+\binom{5}{4} = 1+10+5 = 16 = 2^4\)

Ganjil: \(\binom{5}{1}+\binom{5}{3}+\binom{5}{5} = 5+10+1 = 16 = 2^4\) ✓

05
Teorema Binomial
Inti dari identitas kombinasi

Teorema Binomial memberikan ekspansi dari pangkat dua suku secara umum:

\[(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k\]
Bukti Pembuktian via Induksi Matematika ›
Basis Induksi (n = 1)

\((x+y)^1 = x+y = \binom{1}{0}x + \binom{1}{1}y\) ✓

Hipotesis Induksi

Asumsikan teorema berlaku untuk \(n = m\):

\((x+y)^m = \displaystyle\sum_{k=0}^{m}\binom{m}{k}x^{m-k}y^k\)

Langkah Induksi (n = m+1)

\((x+y)^{m+1} = (x+y)(x+y)^m = (x+y)\displaystyle\sum_{k=0}^m\binom{m}{k}x^{m-k}y^k\)

\(= \displaystyle\sum_{k=0}^m\binom{m}{k}x^{m+1-k}y^k + \sum_{k=0}^m\binom{m}{k}x^{m-k}y^{k+1}\)

Dengan reindeksasi dan identitas Pascal (ID-3) diperoleh:

\(= \displaystyle\sum_{k=0}^{m+1}\binom{m+1}{k}x^{m+1-k}y^k\) ✓

⚡ Kasus-Kasus Khusus Penting

• \(x=y=1\): \(2^n = \displaystyle\sum_{k=0}^n\binom{n}{k}\) (Identitas ID-5)

• \(x=1, y=-1\): \(0 = \displaystyle\sum_{k=0}^n(-1)^k\binom{n}{k}\) (Identitas ID-6)

• \(x=y=\frac{1}{2}\): \(1 = \displaystyle\frac{1}{2^n}\sum_{k=0}^n\binom{n}{k}\)

• \(x=1, y=2\): \(3^n = \displaystyle\sum_{k=0}^n\binom{n}{k}2^k\)

06
Identitas Turunan dari Teorema Binomial
Dengan diferensiasi, integrasi, dan substitusi
Turunan Identitas Jumlah \(k\binom{n}{k}\) ›
\[\sum_{k=0}^{n} k\binom{n}{k} = n\cdot 2^{n-1}\]
Bukti via Diferensiasi

Mulai dari \((1+x)^n = \displaystyle\sum_{k=0}^n\binom{n}{k}x^k\).

Diferensiasi terhadap \(x\): \(n(1+x)^{n-1} = \displaystyle\sum_{k=0}^n k\binom{n}{k}x^{k-1}\)

Substitusi \(x=1\): \(n\cdot 2^{n-1} = \displaystyle\sum_{k=0}^n k\binom{n}{k}\) ✓

Contoh: n = 4

\(0\cdot1 + 1\cdot4 + 2\cdot6 + 3\cdot4 + 4\cdot1 = 0+4+12+12+4 = 32 = 4 \cdot 2^3\) ✓

Turunan Identitas Jumlah \(k^2\binom{n}{k}\) ›
\[\sum_{k=0}^{n} k^2\binom{n}{k} = n(n+1)\cdot 2^{n-2}\]
Bukti via Diferensiasi ke-2

Diferensiasi ke-2 dari \((1+x)^n\), lalu kalikan \(x\) dan substitusi \(x=1\):

\(\displaystyle\sum_{k=0}^n k^2\binom{n}{k} = n(n+1)2^{n-2}\) ✓

Contoh: n = 3

\(0+1\cdot3+4\cdot3+9\cdot1 = 0+3+12+9 = 24 = 3\cdot4\cdot2^1\) ✓

Integral Identitas via Integrasi ›
\[\sum_{k=0}^{n} \frac{\binom{n}{k}}{k+1} = \frac{2^{n+1}-1}{n+1}\]
Bukti via Integrasi

Integrasikan \((1+x)^n = \displaystyle\sum_{k=0}^n\binom{n}{k}x^k\) dari 0 hingga 1:

\(\displaystyle\int_0^1(1+x)^n\,dx = \left[\frac{(1+x)^{n+1}}{n+1}\right]_0^1 = \frac{2^{n+1}-1}{n+1}\)

Sedangkan ruas kanan: \(\displaystyle\int_0^1\sum_{k=0}^n\binom{n}{k}x^k\,dx = \sum_{k=0}^n\frac{\binom{n}{k}}{k+1}\) ✓

Sumar Identitas Penjumlahan Atas (Upper Summation) ›
\[\sum_{i=0}^{n} \binom{i+k}{k} = \binom{n+k+1}{k+1}\]
Kasus Khusus k = 0

\(\displaystyle\sum_{i=0}^n 1 = n+1 = \binom{n+1}{1}\) ✓

Kasus Khusus k = 1

\(\displaystyle\sum_{i=0}^n(i+1) = \frac{(n+1)(n+2)}{2} = \binom{n+2}{2}\) ✓

Contoh: k = 2, n = 4

\(\binom{2}{2}+\binom{3}{2}+\binom{4}{2}+\binom{5}{2}+\binom{6}{2} = 1+3+6+10+15 = 35 = \binom{7}{3}\) ✓

Ekspansi Binomial Interaktif
Ekspansi \((ax + by)^n\) langkah demi langkah
Generator Ekspansi Binomial

Tekan tombol Ekspansi untuk melihat hasilnya.

💡 Catatan Penting

Suku ke-\((k+1)\) dari ekspansi \((x+y)^n\) adalah: \(T_{k+1} = \binom{n}{k}\,x^{n-k}\,y^k\)

Contoh: Suku ke-3 dari \((2x+y)^5\): \(T_3 = \binom{5}{2}(2x)^3(y)^2 = 10 \cdot 8x^3y^2 = 80x^3y^2\)
